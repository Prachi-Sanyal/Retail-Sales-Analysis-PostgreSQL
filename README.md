# Customer Behavior Analysis with PostgreSQL

> **Note:** This project was completed as a guided learning project based on a YouTube tutorial with SQL analysis, a Power BI dashboard, and a Gamma AI presentation to strengthen practical Data Analytics skills.

## Project Overview

This project analyzes customer purchasing behavior using Python, Pandas, PostgreSQL, SQL, and Power BI.

The dataset was imported into PostgreSQL using Python and SQLAlchemy. Analytical SQL queries were performed to extract business insights related to customer spending patterns, subscriptions, discounts, product performance, shipping preferences, and customer segmentation.

The results were further visualized through an interactive Power BI dashboard, and key findings were summarized in a presentation created using Gamma AI.

## Project Workflow

1. Load dataset into Pandas DataFrame
2. Connect Python to PostgreSQL
3. Create and populate database tables
4. Execute analytical SQL queries
5. Extract business insights
6. Create visualizations in Power BI
7. Summarize findings in a presentation

## SQL Analysis Performed

### Customer Revenue Analysis

* Revenue generated by male vs female customers

### Discount Analysis

* Customers who used discounts and still spent above average
* Products with the highest discount usage rate

### Product Performance

* Top-rated products
* Most purchased products by category

### Shipping Analysis

* Average purchase amount by shipping type

### Subscription Analysis

* Spending comparison between subscribers and non-subscribers
* Subscription behavior of repeat customers

### Customer Segmentation

* New Customers
* Returning Customers
* Loyal Customers

### Revenue Analysis

* Revenue contribution across customer age groups
